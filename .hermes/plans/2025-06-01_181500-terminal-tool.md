3. **前端工具组件** — 没有终端/命令行 UI 工具
4. **工具注册** — 需要按 DeskPal 插件规范注册新工具

## 技术方案

### 方案选型：tauri-plugin-shell（推荐）

使用已有的 `tauri-plugin-shell` — Tauri 官方维护，基于 scope 的安全模型，支持：
- `Command.create('cmd', ['/C', '...'])` — 创建子进程
- `.execute()` — 异步执行，返回 stdout/stderr + exit code
- `.spawn()` — 后台执行
- `.stdout.on('data', ...)` — 流式读取（实时输出）
- `.kill()` — 终止进程

**前端 API 示例：**
```ts
import { Command } from '@tauri-apps/plugin-shell';

const cmd = Command.create('cmd', ['/C', userInput]);
const output = await cmd.execute();
// output.stdout, output.stderr, output.code
```

### 不推荐方案：自定义 Rust command

虽然可以手写 `#[tauri::command]` 用 `std::process::Command`，但：
- 需要自己处理编码、超时、杀死进程
- 需要自己管理作用域安全
- tauri-plugin-shell 是 Tauri 官方推荐的”正确方式“

### Step 1 — 创建 capabilities 文件

**文件**：`src-tauri/capabilities/default.json`（新建目录 + 文件）

```json
{
  "identifier": "default",
  "description": "Default capabilities for DeskPal",
  "windows": ["main"],
  "permissions": [
    "core:default",
    "shell:allow-open",
    "shell:allow-execute",
    "shell:allow-spawn",
    "shell:allow-stdin-write",
    "shell:allow-kill"
  ]
}
```

### Step 2 — 配置 shell scope

**文件**：`src-tauri/tauri.conf.json` — 修改 `plugins.shell`

```json
"plugins": {
  "shell": {
    "open": true,
    "scope": [
      {
        "name": "cmd",
        "cmd": "cmd",
        "args": [
          { "validator": "\\S+" }
        ]
      },
      {
        "name": "powershell",
        "cmd": "powershell",
        "args": true
      }
    ]
  }
}
```

**说明**：
- `cmd` scope：Windows 命令提示符，`cmd /C <任意命令>`
- `powershell` scope：PowerShell，`powershell -Command <任意命令>`
- args validator `"\\S+"` 确保至少传入一个非空参数
- 跨平台扩展：macOS/Linux 可追加 `sh` / `bash` scope

#### `utils.ts` — 核心逻辑

```ts
import { Command } from '@tauri-apps/plugin-shell';

export interface ExecutionResult {
  stdout: string;
  stderr: string;
  code: number | null;
  duration: number;
}

/** 执行命令（等待完成，返回全部输出） */
export async function executeCommand(
  command: string,
  signal?: AbortSignal
): Promise<ExecutionResult> {
  const start = Date.now();

  // Windows: cmd /C <command>
  const cmd = Command.create('cmd', ['/C', command]);

  // 支持取消
  signal?.addEventListener('abort', () => {
    cmd.kill();
  });

  const output = await cmd.execute();
  return {
    stdout: output.stdout,
    stderr: output.stderr,
    code: output.code,
    duration: Date.now() - start,
  };
}

/** 执行命令（流式，实时返回输出行） */
export function spawnCommand(
  command: string,
  onStdout: (line: string) => void,
  onStderr: (line: string) => void,
  onExit: (code: number | null) => void,
): () => void {
  const cmd = Command.create('cmd', ['/C', command]);

  cmd.stdout.on('data', (line) => onStdout(line));
  cmd.stderr.on('data', (line) => onStderr(line));

  cmd.spawn().catch((err) => {
    onStderr(`Error: ${String(err)}`);
    onExit(1);
  });

  // 返回 kill 函数
  let resolved = false;
  cmd.on('close', (data) => {
    if (!resolved) {
      resolved = true;
      onExit(data.code);
    }
  });
  cmd.on('error', (error) => {
    if (!resolved) {
      resolved = true;
      onStderr(String(error));
      onExit(1);
    }
  });

  return () => cmd.kill();
}
```

## 安全考量

| 风险 | 缓解措施 |
|------|----------|
| 任意命令执行 | Tauri shell scope 限制为 `cmd` + `powershell`；如需更严格可限制 args validator |
| 恶意命令（`rm -rf`、`del /F /S`） | 工具用途本身就是执行用户输入的命令，由用户自担风险。可在 scope 中进一步白名单化具体命令 |
| CSP 限制 | `default-src 'self'` 不受影响，shell plugin 通过 IPC 通信 |
| 跨平台 | 当前 scope 仅 Windows；可扩展 `sh`/`bash` scope |
